Sudoku Puzzle
=============================================================
Written by: Ron Efraim, id = 204661425, login = ronef

The program receives from user numbers of row, column and then the value. 
If the user does not enter a number then the cell remains with zero.
the program puts a number between 1-4 in empty cells and checks for each number if there is no similar
number in the row, column, and cube.

Compile: g++ -Wall ex9a.cc –o ex9a
Run: ex9a

Input:
 a. row, to end the input, put -1 in row.
 b. column
 c. value
 
Output:
 a. print the complete sudoku board.

Gold Bars
=============================================================
Writen by: Ron Efraim, id = 204661425, login = ronef

Compile: g++ -Wall ex9c.cc –o ex9c
Run: ex9c

input:
